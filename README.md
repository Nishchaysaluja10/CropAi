CropAI is an AI-powered, multilingual web platform designed to provide comprehensive agricultural intelligence to farmers. Developed as a solution for the **Smart India Hackathon (SIH)**, this tool aims to empower farmers by providing actionable insights on yield prediction, disease detection, weather, market prices, and government schemes, helping them increase productivity and profitability.
